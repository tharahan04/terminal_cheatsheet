# Terminal Cheatsheet 

* `mkdir` - create new folder
* `touch` create new file
* `cd` - change directory 
* `ls` - list folders and files
* `mv` - moving files and folders
* `rm` - removing files and folders
* `pwd` - print working directory (to see where you are)
